# Spidey Social

Short-lived activity posts. People can swing in, connect, and the web disappears later.

## Project Structure

| Directory   | Contents                                                                 |
|-------------|---------------------------------------------------------------------------|
| `frontend/` | React + Vite + Tailwind app. Noir shell UI, API helper scaffold.         |
| `infra/`    | AWS CDK TypeScript. SpideySocialStack: DynamoDB, Cognito, S3, CloudFront, API Gateway + Lambda. |

See [docs/SETUP.md](docs/SETUP.md) for local verify steps and optional deploy.
